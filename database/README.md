# Database Setup

1. Create a MySQL database named `ecommerce_db`.
2. Run `schema.sql` to create tables, indexes, and triggers.
3. (Optional) Run `seed.sql` or `update_products_final.sql` to insert or update sample data.
4. Make sure your `.env` file is properly configured with `DB_HOST`, `DB_USER`, `DB_PASSWORD`, and `DB_NAME`.
